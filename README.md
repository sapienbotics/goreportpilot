# ReportPilot

AI-powered client reporting for digital marketing agencies and freelancers.

## What it does
- Connects to Google Analytics, Meta Ads, and Google Ads via OAuth
- Pulls performance data automatically
- AI (GPT-4o) writes narrative insights explaining trends and recommendations
- Generates branded reports in PowerPoint and PDF
- Delivers reports to clients via email on a schedule

## Tech Stack
- **Frontend:** Next.js 14 + TypeScript + Tailwind CSS
- **Backend:** Python + FastAPI
- **Database:** Supabase (PostgreSQL)
- **AI:** OpenAI GPT-4o
- **Reports:** python-pptx + ReportLab
